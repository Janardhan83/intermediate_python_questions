## 📌 Project Overview

The **Student Performance Analyzer** processes student marks data and generates useful academic insights.

The program analyzes student performance across multiple subjects and provides:

* Total marks of each student
* Average marks calculation
* Overall top scorer identification
* Subject-wise topper analysis
* Grade assignment
* Complete performance report generation

---

## 📂 Dataset Structure

Each record contains:

```python
(student_name, subject, marks)
```

Example:

```python
students = [
    ("Rahul", "Math", 85),
    ("Anu", "Math", 92),
    ("Kiran", "Math", 78),
    ("Rahul", "Science", 88),
    ("Anu", "Science", 95),
    ("Kiran", "Science", 82),
    ("Rahul", "English", 75),
    ("Anu", "English", 89),
    ("Kiran", "English", 91)
]
```

---

## 🚀 Features

### 1️⃣ Total Marks Calculation

Calculates total marks obtained by each student.

Example:

```text
Rahul -> 248
Anu -> 276
Kiran -> 251
```

---

### 2️⃣ Average Marks Calculation

Calculates the average score for each student.

Example:

```text
Rahul -> 82.67
Anu -> 92.00
Kiran -> 83.67
```

---

### 3️⃣ Top Scorer Detection

Identifies the highest-scoring student.

Example:

```text
Anu -> 276
```

---

### 4️⃣ Subject-wise Topper Analysis

Finds the highest scorer in every subject.

Example:

```text
Math -> Anu (92)
Science -> Anu (95)
English -> Kiran (91)
```

---

### 5️⃣ Grade Assignment

Grades are assigned using the following criteria:

| Average Marks | Grade |
| ------------- | ----- |
| 90 and above  | A     |
| 80 - 89.99    | B     |
| 70 - 79.99    | C     |
| Below 70      | D     |

---

### 6️⃣ Final Performance Report

Generates a structured student performance report.
