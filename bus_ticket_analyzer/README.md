# Smart Bus Ticket Analyzer 🚍

## Project Overview

The Smart Bus Ticket Analyzer is a Python program that processes passenger travel data and generates useful insights from bus trip records.

This project demonstrates the use of Python data structures and logical problem-solving techniques to analyze travel information.

---

## Problem Statement

Given a list of bus trips containing:

* Passenger Name
* Source City
* Destination City
* Ticket Price

The program performs the following analyses:

1. Calculate the total amount spent by each passenger.
2. Find the passenger who traveled the most times.
3. Identify the most expensive trip.
4. Find all unique cities involved in the trips.
5. Generate a complete travel report.
